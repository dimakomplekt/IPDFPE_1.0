# Inventor PDF pocket export

Три макроса, предназначенных для пакетного эспорта чертежей формата .idw в формат .pdf:

1) [Пакетный эспорт всех отдельных фалов из всех подпапок чертежей проекта](https://github.com/dimakomplekt/IPDFPE_1.0/blob/main/src/pdf_pocket_export_all.bas)
2) [Пакетный экспорт файлов из конкретной подпапки чертежей проекта](https://github.com/dimakomplekt/IPDFPE_1.0/blob/main/src/pdf_pocket_export_current_subfolder.bas)
3) [Пакетный экспорт всех листов какого-либо отдельного чертежа (для структуры ведения КД изделия в 1 файле)](https://github.com/dimakomplekt/IPDFPE_1.0/blob/main/src/pdf_pocket_export_one_file.bas)


Подробное описание макросов доступно в [description](https://github.com/dimakomplekt/IPDFPE_1.0/blob/main/description)


## Запуск 

Макрос 1 запускается из вспомогательного пустого idf-файла с одним скопированным правилом в корневой папке ред.формата чертежей.

```

КОРЕНЬ_РЕД_ФОРМАТ
|
├── pocket_export.idw (запускать макрос тут)
|
├── 1_Архив   ← игнорим
|
├── Пакет_1
│   ├── файл1.idw
│   ├── файл2.idw
│
├── Пакет_2
│   ├── файл3.idw
|
КОРЕНЬ_PDF_ФОРМАТ (где PDF раскладываются по подкаталогам, дублирующим структуру РФ)

```


Макрос 2 запускается из сборочного чертежа текущего подкаталога (idf-файла) с скопированным правилом в каком-либо подкаталоге корневой папке ред.формата чертежей.

```

КОРЕНЬ_РЕД_ФОРМАТ
|
├── pocket_export.idw
|
├── 1_Архив
|
├── Пакет_1
│   ├── файл1_сборочный_чертеж.idw (запускать макрос тут)
│   ├── файл2.idw
│
├── Пакет_2
│   ├── файл3.idw
|
КОРЕНЬ_PDF_ФОРМАТ (где PDF раскладываются по подкаталогам, дублирующим имена папок из которых был произведен вызов)

```

Макрос 3 запускается из любого отдельного файла

```

КОРЕНЬ_РЕД_ФОРМАТ
|
├── 1_Архив
|
├── Пакет_1 
│   ├── файл1_комплект_чертежей.idw (где запускаешь макрос)
│
├── Пакет_2
│   ├── файл3.idw
|
КОРЕНЬ_PDF_ФОРМАТ (где PDF раскладываются без подкаталога, по с именем, дублирующим имя файла)

```
